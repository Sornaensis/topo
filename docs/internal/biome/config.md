# Biome Configuration

> **Module:** `Topo.BiomeConfig` (364 LOC)
> **Status:** Stub

## Overview

`BiomeConfig` bundles classification rules, thresholds, refinement
config, and vegetation config.

## Named Presets

| Preset | Character |
|--------|-----------|
| Default | Balanced Earth-like distribution |
| `aridBiomeConfig` | Expanded desert/savanna zones |
| `lushBiomeConfig` | Expanded forest/rainforest zones |

## Pipeline Stage

`classifyBiomesStage` — runs biome classification over the full terrain.
