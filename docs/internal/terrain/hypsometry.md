# Hypsometry

> **Module:** `Topo.Hypsometry` (210 LOC)
> **Status:** Stub

## Overview

Hypsometric elevation redistribution. Remaps the elevation probability
density toward low values, mimicking Earth's hypsometric curve (~70%
of land below 500m).

## Key Properties

- Strictly monotonic: preserves drainage topology
- Does not create new peaks or valleys
- Operates globally on the full elevation grid
