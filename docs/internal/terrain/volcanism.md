# Volcanism

> **Module:** `Topo.Volcanism` (301 LOC)
> **Status:** Stub

## Overview

Volcanic vent generation. Places volcanic features based on plate
boundary proximity and hotspot locations.

## Vent Placement

- **Boundary-weighted** — higher probability near convergent boundaries
- **Hotspot** — random intra-plate volcanic activity

## Effects

- Eruption thresholds and magma recharge cycles
- Deposit effects: elevation modification, soil enrichment
- Per-vent: eruption state, deposit radius, composition
